# 通讯录管理系统 - 使用说明

## 项目概述
这是一个前后端分离的通讯录管理系统，实现了联系人的添加、编辑、删除和搜索功能。

## 项目结构
```
软件工程作业1/
  ├── contacts_frontend/   # 前端项目
  └── contacts_backend/    # 后端项目
```

## 后端运行指南

### 1. 安装依赖
```bash
cd contacts_backend
pip install -r requirements.txt
```

### 2. 启动后端服务
```bash
cd src
python app.py
```

- 服务默认运行在 http://localhost:5000
- 数据库会自动创建（SQLite格式）
- 开启了CORS支持，允许前端跨域访问

## 前端运行指南

### 1. 直接打开HTML文件
由于前端使用纯静态文件实现，可以直接在浏览器中打开：
```
contacts_frontend/src/contacts.html
```

### 2. 使用Web服务器（推荐）
为了更好的体验，可以使用简单的Web服务器：

**使用Python的内置服务器：**
```bash
cd contacts_frontend/src
python -m http.server 8000
```

然后在浏览器中访问 http://localhost:8000/contacts.html

## 功能使用

### 添加联系人
1. 在表单中填写联系人信息（姓名和电话为必填项）
2. 点击"保存"按钮

### 编辑联系人
1. 在联系人列表中找到要编辑的联系人
2. 点击该联系人的"编辑"按钮
3. 在表单中修改信息
4. 点击"保存"按钮

### 删除联系人
1. 在联系人列表中找到要删除的联系人
2. 点击该联系人的"删除"按钮
3. 在确认对话框中点击"确定"

### 搜索联系人
1. 在搜索框中输入关键词（姓名或电话）
2. 联系人列表会实时更新，显示匹配的联系人

## 注意事项

1. 确保后端服务正在运行，否则前端将无法正常保存数据（但会显示模拟数据用于演示）
2. 电话号码在系统中是唯一的，不能重复添加
3. 邮箱和地址为可选字段
4. 所有数据保存在SQLite数据库中，重启服务后数据不会丢失
